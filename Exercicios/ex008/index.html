Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Nessa frase temos um termo em destaque usando a tag STRONG

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântica).

Nesta frase temos um termo em ênfase usando a tag EM (semântica)

Texto marcado

Podemos também criar um texto marcado usando a tag MARK

Texto grande e pequeno

Podemos criar um texto grande usando a tag BIG (obsoleta) e um texto pequeno usando a tag SMALL (ainda em uso)

Texto deletado

Podemos marcar um texto excluído usando a tag DEL

Texto inserido

Podemos marcar um texto como inserido usando a tag INS

Lembrando que também é possível sublinhar o texto utilizando a tag U, porém ela não é semântica.

Texto sobrescrito e subescrito

Podemos escrever coisas como 2033+5 com a tag SUP

Podemos escrever coisas como H2O com a tag SUB
